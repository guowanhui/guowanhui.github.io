若是因为经济拮据不得已可使用以下方法：

2020-05-05

  • 先下载压缩包解压后得到jetbrains-agent.jar，把它放到你认为合适的⽂件夹内。
    下载⻚⾯：https://pan.baidu.com/s/1P9etzeDTYqFaBtsu_st5og 提取码：qsw0

  • 启动你的pycharm，如果上来就需要注册，选择：试⽤（Evaluate for free）进⼊IDE。

  • 点击你要注册的IDE菜单： Configure 或 Help -> Edit Custom VM Options ...
    如果提示是否要创建⽂件，请点 Yes 。

  • 在打开的vmoptions编辑窗⼝末⾏添加： -javaagent:/absolute/path/to/jetbrains-agent.jar⼀定要⾃⼰确认好路径(不要使⽤中⽂路径)，填错会导致IDE打不开！！！最好使⽤绝对路径

⼀个vmoptions内只能有⼀个 -javaagent 参数。
示例:
mac: -javaagent:/Users/neo/jetbrains-agent.jar
linux: -javaagent:/home/neo/jetbrains-agent.jar
windows: -javaagent:C:\Users\neo\jetbrains-agent.jar
如果还是填错了，参考这篇⽂章编辑vmoptions补救：
https://intellij-support.jetbrains.com/hc/en-us/articles/206544519

  • 重启你的pycharm。

  • 点击pycharm菜单 Help -> Register... 或 Configure -> Manage License...


  • ⽀持两种注册⽅式：License server 和 Activation code:

  • 选择License server⽅式，地址填⼊： http://jetbrains-license-server 或者点击按钮： Discover Server 来⾃动填充地址。

  • 选择Activation code⽅式离线激活，请使⽤： ACTIVATION_CODE.txt 内的注册码激活。 !

    如果激活窗⼝⼀直弹出（error 1653219），请去hosts⽂件⾥移除jetbrains相关的项⽬。如果你需要⾃定义License name，请访问：ttps://zhile.io/custom-license.html
